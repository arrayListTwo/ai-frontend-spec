# Component Templates

本目录包含 Vue3 + Element Plus + RuoYi-Vue3 的组件模板。

## 模板列表

| 模板 | 说明 |
|------|------|
| form-dialog/ | 弹窗表单组件 |
| search-form/ | 搜索表单组件 |
| table-toolbar/ | 表格工具栏组件 |
| upload-dialog/ | 上传弹窗组件 |
| tree-selector/ | 树形选择器组件 |

## 使用方式

1. 根据组件功能选择合适的模板
2. 复制模板文件到项目对应目录
3. 根据实际需求修改模板内容
4. 确保符合 AFES 编码规范
