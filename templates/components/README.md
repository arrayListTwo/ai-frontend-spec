# Component Templates

本目录包含 Vue3 + Element Plus + RuoYi-Vue3 的组件模板。

## 模板列表

| 模板 | 说明 |
|------|------|
| search-form/ | 搜索表单组件 |
| data-table/ | 数据表格组件 |
| edit-dialog/ | 编辑弹窗组件 |
| form-section/ | 表单分区组件 |
| upload-panel/ | 上传面板组件 |

## 使用方式

1. 根据组件功能选择合适的模板
2. 复制模板文件到项目对应目录
3. 根据实际需求修改模板内容
4. 确保符合 AFES 编码规范
