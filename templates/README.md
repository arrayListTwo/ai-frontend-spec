# Templates

本目录包含 Vue3 + Element Plus + RuoYi-Vue3 的代码模板。

## 目录

```
templates/
├── pages/              # 页面模板
│   ├── crud-page/      # CRUD 增删改查页面
│   ├── tree-page/      # 树形结构页面
│   └── README.md
├── components/         # 组件模板
│   ├── form-dialog/    # 弹窗表单组件
│   ├── search-form/    # 搜索表单组件
│   ├── table-toolbar/  # 表格工具栏组件
│   ├── upload-dialog/  # 上传弹窗组件
│   ├── tree-selector/  # 树形选择器组件
│   └── README.md
└── README.md
```

## 使用方式

参考对应目录下的 README 和模板文件，根据实际需求调整后使用。
