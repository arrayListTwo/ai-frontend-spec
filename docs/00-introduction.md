# 规范介绍

AFES（AI Frontend Engineering Specification）是一套专为 AI 辅助编码场景设计的前端工程规范，适用于 Vue3 + Element Plus + RuoYi-Vue3 技术栈。

本规范定义了 AI 在前端开发中的角色、原则、流程和规则，确保 AI 生成的代码具备生产级质量。它不是通用的前端编码规范，而是一套 **AI 专用的编码协议**。

## 适用范围

- 技术栈：Vue 3 + Element Plus + RuoYi-Vue3
- 场景：AI 辅助编码（Cursor、GitHub Copilot、ChatGPT 等）
- 目标用户：前端开发者、AI 工具使用者、团队技术负责人

## 规范结构

本规范共 16 章，按逻辑分为四个部分：

1. **基础定义**（01-02）：角色定位与工作原则
2. **工作方法**（03-05）：上下文感知、决策规则、工作流程
3. **编码规则**（06-11）：编码、页面、组件、API、样式、质量
4. **输出与约束**（12-15）：自检、Prompt、输出、禁止规则
